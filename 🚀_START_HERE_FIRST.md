## 🔧 If You Need Help

### **Common Issues:**

**Backend won't start?**
```bash
cd backend
rm -rf node_modules
npm install
npm run dev
```

**AI Service errors?**
```bash
cd ai-service
pip install --upgrade pip
pip install -r requirements.txt
```

**Port conflicts?**
```bash
lsof -i :5000  # Find process
kill -9 <PID>  # Kill it
```

**MongoDB connection?**
- Your URI is already configured
- Just make sure you have internet connection
